# Syntax

'Cool' syntax can lead to actual language features and abstractions.


## bitwise not

`~`: bitwise not operator

    ~[1, 2, 3].indexOf(x)

is equivalent to

    [1, 2, 3].indexOf(x) != -1


## logical operators

Using `&&` and `||`

instead of:

    if (x) x.y()

write:

    x && x.y()
